Mathgenerator/Todarith

My original goal for the todarith/mathgenerator projects were to create a website that contains a large database full of math problems, and then train a computer to give similar problems and their solutions to a problem that the user gave. I had no experience with machine learning and decided to start generating problems. I set up a github repo and talked about my project online during Digital Ocean's Hacktoberfest. Mathgenerator was created in Python, with an object oriented approach that created a Generator instance for every generator function, of which there were around 110. You can find the project here on Github Todarith is the website that I created to accompany the generator. I ended up abandoning the machine learning aspect of the project, but would still like to try to create it once I have the time and knowledge of how to create it. You can find the site at todarith.ml and a working demo of mathgenerator at todarith.ml/generate
